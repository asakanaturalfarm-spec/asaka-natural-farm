Frequently Asked Questions

注文について
Q 最低注文金額はありますか？
+
4,500円以上（税込・送料別）から注文可能です。
少量配送による品質劣化を防ぐため、最低注文金額を設定しています。
Q 注文のキャンセルはできますか？
+
発送前であれば可能です。
キャンセルをご希望の場合は、お早めにご連絡ください。
発送後のキャンセルはお受けできません。
詳細は返品・キャンセルポリシーをご確認ください。
Q 会員登録は必要ですか？
+
はい、必要です。
商品の購入には会員登録（無料）が必要です。
注文履歴の確認、配送先情報の保存など便利な機能がご利用いただけます。
商品について
Q どのような栽培方法ですか？
+
自然農法（無農薬・無肥料）で栽培しています。
化学肥料・農薬を一切使用せず、自家採種による種を使用しています。

詳細な栽培方法については、安積自然農園ホームページをご覧ください。
Q 有機JAS認証はありますか？
+
取得していません。
有機JAS認証は取得していませんが、無農薬・無肥料の自然農法で栽培しています。
認証取得には多額の費用がかかるため、その分を価格に反映させることを優先しています。
Q 商品の品質保証はありますか？
+
商品到着後24時間以内に、破損・腐敗等の不良があった場合は、
全額返金または代替品の再送にて対応いたします。
詳細は返品・キャンセルポリシーをご確認ください。
Q 野菜の大きさや形はどうですか？
+
自然農法のため、サイズや形にばらつきがあります。
スーパーのように規格化された形状ではありませんが、味や栄養価に問題はありません。
多少の色ムラ、表面の傷は農産物の特性としてご理解ください。
配送について
Q 送料はいくらですか？
+
全国一律 500円（ヤマト運輸クール便）です。
鮮度を保つため、冷蔵便で配送します。
Q 発送はいつですか？
+
月曜日・水曜日・金曜日のいずれかに発送します。
ご注文確定後、最短で翌営業日に発送いたします。
お届け日は発送日の翌日～3日後（地域により異なる）となります。
Q 配達時間帯の指定はできますか？
+
可能です。以下の時間帯から選択できます。
・午前中（8:00～12:00）
・14:00～16:00
・16:00～18:00
・18:00～20:00
・19:00～21:00
・指定なし
Q 配送状況の確認はできますか？
+
発送後、お問い合わせ番号（追跡番号）をメールでお知らせします。
ヤマト運輸の荷物追跡サービスで配送状況をご確認いただけます。
支払いについて
Q どの支払方法が使えますか？
+
以下の支払方法に対応しています：
・クレジットカード（VISA、Mastercard、JCB、AMEX、Diners）
・コンビニ決済（セブンイレブン、ローソン、ファミリーマート）
・PayPay
・LINE Pay
・銀行振込（前払い）
・代金引換
Q 手数料はかかりますか？
+
・クレジットカード、PayPay、LINE Pay：無料
・コンビニ決済：無料
・代金引換：330円
・銀行振込：振込手数料はお客様負担
Q 領収書は発行できますか？
+
可能です。
ご希望の場合は、注文時の備考欄にその旨をご記入ください。
商品に同梱、またはPDFでメール送付いたします。
