このフォルダには東北大学などの研究機関から受領するレポートと関連データを置いてください。

推奨ファイル構成:

- evidence-report.pdf          // 受領したフルレポート（PDF）
- dataset.csv                  // 解析で使った集計データ
- metadata.json                // データのカラム説明・単位・採取日など

注意:
- 公開可能な資料のみ配置してください。個人情報を含むデータは適切にマスク/匿名化してください。
- レポートをアップロードしたら、サイト内の「研究・エビデンス」セクションからリンクが有効になります。
